Supported Routes

  • /mine

    Mines a new block and returns the block

  • /transactions/new

    Requires:

                        
                        {
                            'sender': '<sender address>',
                            'recipient': '<recipient address>',
                            'amount': <amount to send>
                        }

    Makes a new transaction

  • /chain

    Returns the current chain

  • /nodes/register

    Requires:

                        
                        {
                            'nodes': [
                                '<Node address 1>',
                                '<Node address 2>',
                                ...,
                                '<Node address N>'
                            ]
                        }

    Registers new nodes to be part of the blockchain consensus

  • /nodes/resolve

    Attempts to resolve conflicts with registered nodes
